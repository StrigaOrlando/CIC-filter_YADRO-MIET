# Golden model

Папка содержит текущую MATLAB golden model CIC-фильтра

Модель используется как эталон для проверки RTL-реализации

## Содержимое


- MATLAB-файлы golden model
- вспомогательные функции модели
- скрипты генерации тестовых входов
- скрипты сравнения RTL-результата с MATLAB-эталоном
- инструкция по запуску текущей версии модели

## Назначение модели

Golden model формирует тестовые данные для RTL testbench и эталонный результат для сравнения

Основные файлы, которые используются при проверке:

```text
input.txt            — входные данные для RTL testbench
expected_output.txt  — эталонный выход, рассчитанный в MATLAB
rtl_output.txt       — результат RTL-симуляции
```

## Общий порядок проверки

1. Запустить MATLAB golden model.
2. Сформировать `input.txt` и `expected_output.txt`.
3. Запустить RTL testbench на данных из `input.txt`.
4. Получить `rtl_output.txt`.
5. Сравнить `rtl_output.txt` с `expected_output.txt`.

Подробный порядок запуска описан в инструкции к текущей версии модели.

## Примечание

Текущая версия модели является базовой и может дорабатываться вместе с RTL-реализацией.
